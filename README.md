## ✨ Sobre o Projeto

Este é meu portfolio pessoal, desenvolvido para apresentar meus projetos, habilidades e experiência como Desenvolvedor Full Stack. O site conta com animações suaves, design moderno e responsivo, e uma experiência de usuário intuitiva.

### 🎯 Funcionalidades

- 🏠 **Página Inicial** - Apresentação profissional com estatísticas animadas
- 💼 **Serviços** - Descrição dos serviços oferecidos (Dev Web, UI/UX, Branding, SEO)
- 📋 **Currículo** - Experiências, educação, habilidades e sobre mim
- 🖥️ **Projetos** - Portfólio de projetos com links para demos e repositórios
- 📧 **Contato** - Formulário de contato funcional com EmailJS

## 🚀 Começando

### Pré-requisitos

- Node.js 18.x ou superior
- npm, yarn, pnpm ou bun

### Instalação

```bash
# Clone o repositório
git clone https://github.com/alexandrinodev/portfolio.git

# Entre no diretório
cd portfolio

# Instale as dependências
npm install
# ou
yarn install
# ou
pnpm install
# ou
bun install
```

### Executando em Desenvolvimento

```bash
# Inicie o servidor de desenvolvimento
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

### Build para Produção

```bash
# Crie a build de produção
npm run build

# Inicie o servidor de produção
npm run start
```
